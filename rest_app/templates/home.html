API Endpoints

User Registration and Token Generation Endpoints

Endpoint Method Purpose
/auth/users POST Creates a new user with name, email, and password
/auth/users/me GET Displays only the current user
/auth/token/login POST Generates access tokens that can be used in other API calls in this project

Menu Items Endpoints

Endpoint Role Method Purpose
/menu-items Customer, Delivery Crew GET Lists all menu items
/menu-items/{menuItem} Customer, Delivery Crew GET Lists single menu item
/menu-items Manager GET, POST Lists all menu items. Creates a new menu item.
/menu-items/{menuItem} Manager GET, PUT, PATCH Lists single menu item. Updates single menu item. Deletes menu item

User Group Management Endpoints (Manager Role Only)

Endpoint Method Purpose
/groups/manager/users GET, POST Returns all managers. Assigns the user in the payload to the manager group.
/groups/manager/users/{userId} DELETE Removes this particular user from the manager group.
/groups/delivery-crew/users GET, POST Returns all delivery crew. Assigns the user in the payload to delivery crew group.
/groups/delivery-crew/users/{userId} DELETE Removes this particular user from the manager group.

Cart Management Endpoints (Customer Role)

Endpoint Method Purpose
/cart/menu-items GET, POST, DELETE Returns current items in the cart for the current user token. Adds the menu item to the cart. Deletes all menu items .

Order Management Endpoints

Endpoint Role Method Purpose
/orders Customer GET, POST Returns all orders with order items created by this user. Creates a new order by getting current cart items from the cart endpoints and adds those items to the order items table. Then deletes all items from the cart for this user
/orders/{orderId} Customer GET, PUT, PATCH Returns all items for this order id. Updates the order.
/orders Manager GET Returns all orders with order items by all users
/orders/{orderId} Manager DELETE Deletes the order
/orders Delivery Crew GET Returns all orders with orders assigned to the delivery crew.
/orders/{orderId} Delivery Crew PATCH A delivery crew person can only update the order status to 0 or 1.
